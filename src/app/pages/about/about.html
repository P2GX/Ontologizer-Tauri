The is a controlled vocabulary that describes functions, roles, and locations of gene products. A widespread application is the identification of annotation-enriched GO terms in a list of genes that share some biological characteristic, the so-called study set (e.g., genes that are overexpressed in an experiment), compared to a larger list of genes, the population set (e.g., all genes measured in the experiment). These terms are often interpreted as representing the salient biological features of the genes in the study set.

This application is the third edition of the Ontologizer, following in 2004 and in 2008, both of which were created in Java. Ontologizer V3 was created as a Tauri application that makes use of an efficient backend in Rust and a modern frontend that uses the Angular framework.

  4. Population set: All gene symbols from the examined population, one to a line
